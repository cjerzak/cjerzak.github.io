# cjerzak.github.io
Web connector for connorjerzak.com.

**Local preview**
- `python3 -m http.server 4173`
- Open `http://127.0.0.1:4173/`

**Key routes**
- `/` (home)
- `/projects/` (projects + bibliography)
- `/the-lab/` and `/the-lab/services/`
- `/contact/`

**Research bibliography**
